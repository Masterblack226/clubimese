Tout ce que vous devez savoir sur l'abonnement VIP

Le paiement est simple et sécurisé. Sélectionnez votre opérateur (Orange Money ou Moov Money), entrez votre numéro de téléphone et confirmez le paiement via votre code PIN. L'activation de votre compte VIP est immédiate après confirmation du paiement.

Oui, absolument. Vous pouvez annuler votre abonnement VIP à tout moment depuis votre tableau de bord. Vous conserverez l'accès aux avantages VIP jusqu'à la fin de votre période de facturation en cours. Aucune pénalité d'annulation n'est appliquée.

Nous offrons une période de grâce de 7 jours après l'échéance. Vous recevrez des rappels par SMS et email. Si le paiement n'est pas effectué dans les 7 jours, votre accès VIP sera suspendu mais vos données seront conservées. Vous pourrez réactiver votre compte en effectuant le paiement.

Les corrections des devoirs sont généralement publiées dans les 48h suivant la date limite de soumission. Les membres VIP reçoivent une notification par email dès qu'une nouvelle correction est disponible. Toutes les corrections passées sont également accessibles dans votre bibliothèque personnelle.

Oui, nous offrons une garantie satisfaction de 30 jours pour les nouveaux abonnés VIP. Si vous n'êtes pas entièrement satisfait des services VIP dans les 30 premiers jours, nous vous rembourserons intégralement via Mobile Money. Contactez notre support pour initier le remboursement.
